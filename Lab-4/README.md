# Weather Data Visualizer

This project analyzes weather data to produce insights regarding temperature trends, rainfall patterns, and humidity correlations.

## Structure
- `weather_dataset.csv`: Raw input data.
- `cleaned_weather_dataset.csv`: Processed data (generated by script).
- `weather_data_visualizer.py`: Main Python script for analysis and plotting.
- `plots/`: Directory containing generated visualizations.

## How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the script:
   ```bash
   python weather_data_visualizer.py
   ```
3. Check the `plots/` directory for results.
